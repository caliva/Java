BloomboxClient(settings: Settings, target: ClientTarget = ClientTarget.LOCAL)

Specifies a unified Bloombox API client, that is capable of calling methods on any service exposed by the Bloombox Cloud Platform. At the time of writing, this includes:

  • shop/v1: Shop service. For submitting orders, verifying customers, enrolling customers, etc.
  • telemetry/v1beta3: Telemetry service. For submitting telemetry event data of different kinds.
